# Информационная архитектура онлайн‑сервиса для работы с промптами

## Описание проекта

Онлайн‑сервис предназначен для упрощения работы пользователей с
языковыми моделями (LLM) в различных приложениях. Сервис позволяет
создавать, редактировать, хранить и делиться промптами, изучать лучшие
практики их написания, а также использовать готовые шаблоны.

Основные функции системы: 
- редактор промптов с подсветкой синтаксиса;
- база знаний по форматированию промптов;
- база исследований;
- библиотека готовых шаблонов;
- личный кабинет пользователя;
- создание собственных шаблонов;
- публичный каталог промптов (промпт‑хаб);
- добавление промптов в избранное.

# Дерево страниц (информационная архитектура)

## 1. Главная страница

**Цель страницы** - Познакомить пользователя с сервисом. - Дать быстрый
доступ к основным функциям. - Показать популярные и новые промпты.

**Типы контента** - описание сервиса - блок популярных промптов - блок
новых шаблонов - кнопки регистрации и входа - поиск по промптам

**Точки входа** - прямой переход по URL - переход из поисковых систем -
переход по ссылке из социальных сетей

**Точки выхода** - страница входа - страница регистрации - каталог
промптов - база знаний - база исследований

**Цель страницы** - Создание нового аккаунта пользователя.

**Типы контента** - форма регистрации - поля: email, пароль,

**Типы контента** - форма входа - поля: email и пароль - кнопка входа -

**Типы контента** - информация о пользователе - список созданных
